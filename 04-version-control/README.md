# Version Control MCPs

Colección de MCPs para plataformas de control de versiones y colaboración.

## Incluye

- `mcp-github`
- `mcp-gitlab`
- `mcp-bitbucket`
- `mcp-azure-devops`

## Casos de uso

- automatizar issues, PRs y pipelines
- consultar repositorios y estados de CI
- integrar revisiones con agentes y flujos MCP
